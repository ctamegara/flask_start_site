# flask_start_site

Une base pour commencer une application web sur <b> Pythonanywhere </b>, avec <b> Flask </b>.

1) Une fois que vous avez créé votre compte et votre appli Flask, dont le code va dans le dossier mysite , importez dans celui ci les fichiers proposés ici (en respectant l'arborescence).

2) Créez une base de données appelée <b>mydb</b>.

3) Dans le fichier mysite/flask_app.py, modifiez les lignes 19 et 20 en y incluant votre username et le mot de passe de votre base de données.

4) Cliquez sur le bouton run, et dans la console, saisissez la commande "db.create_all()"

Et voilà !

Les utilisateurs peuvent dès lors
- s'inscrire à l'adresse XXX.pythonanywhere.com/signup , 
- se loguer à l'adresse XXX.pythonanywhere.com/login , 
- et se déloguer à l'adresse XXX.pythonanywhere.com/logout. 

Après, c'est à vous de jouer!
